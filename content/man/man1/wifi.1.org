#+TITLE: Manpages - wifi.1
* NAME
wifi - enable/disable internal Wi-Fi device

* SYNOPSIS
*wifi [/command/]*

* COMMANDS
- *on* :: Switch device on.

- *off* :: Switch device off.

- *toggle* :: Toggle device state.

- <none> :: Show device state.

- *--version* :: Print TLP version.
