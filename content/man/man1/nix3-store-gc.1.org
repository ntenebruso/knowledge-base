* Name
=nix store gc= - perform garbage collection on a Nix store

* Synopsis
=nix store gc= [/option/...]

* Examples
- Delete unreachable paths in the Nix store:

#+begin_example
# nix store gc
#+end_example

- Delete up to 1 gigabyte of garbage:

#+begin_example
# nix store gc --max 1G
#+end_example

* Description
This command deletes unreachable paths in the Nix store.
